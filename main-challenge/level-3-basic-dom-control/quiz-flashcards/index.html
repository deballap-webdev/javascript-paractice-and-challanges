let allows you to reassign a variable's value later in you code, whereas const creates a read-only reference that cannot be reassigned after it is declared.

localStorage is a web storage API that allows you to save key-value data directly in the user's browser. This data has no expiration date and remains intact even if the page is refreshed or the browser is closed
